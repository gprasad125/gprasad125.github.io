IPL Run Predictor

May 2022 - June 2022

This project took publically-available statistics from the first 13 seasons of the Indian Premier League (IPL), the world's biggest franchise T20 cricket tournament, and aimed to build a model that would predict the number of runs a batsman would score against any one bowler in a match.

The data processing and modeling was done inside a Jupyter Environment, while presentation of findings was conducted through a Flask dynamic web deployment.

This project was conducted as a "crash-course" project prior to the start of my AI Camp mentorship. You can read about it @ the following GitHub repo.

Soccer Pass Analysis

January 2022 - June 2022

This project took public soccer data collected by StatsBomb from various matches around the world over a set time period. The goal was to explore how teams passed and whether successful passes could be predicted, with varying defintions of what "success" truly was.

The data processing and modeling was done inside a Jupyter Environment, while the two interactive demos were conducted through a multi-page Streamlit webpage run locally.

This project was conducted as part of the 2022 Projects Committee for the Data Science Student Society (DS3) at UC San Diego. You can read about it @ the following GitHub repo. You can also see our project poster, presented in June 2022, below.
